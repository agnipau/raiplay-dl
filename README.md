# raiplay-dl

CLI per scaricare video da RaiPlay. L'eseguibile non richiede dipendenze.

## Istruzioni

```bash
cargo b
cargo r -- 'https://www.raiplay.it/video/2019/10/Il-Collegio-4-6f9681db-62ff-4094-8272-7f5babaebc29.html'
```
